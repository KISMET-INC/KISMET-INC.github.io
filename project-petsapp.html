Project information

  • Name: PetsApp
  • Category: Full-Stack | UX / UI | Mobile Development
  • Tech: JAVA | SQL | GitHub
  • + View on GitHub

A mobile app for pet owners that stores pet information.

  • Created a mobile application using Android Studio and written in Java which aids pet owners by storing pet information such as birthday, age, notes and photos of vet records in a SQLite database for quick easy access.

  • Wrote SQL join queries to implement one to many relationships between each pet and their multiple vet records making the information organized and easily accessible, optimizing application performance.

  • Designed the pet object model in Java and implemented getters and setters to pass the pet object around the app and access the data within it efficiently while ensuring the data stays secure.

  • Wrote the code in Java, a machine independent language, giving the codebase incredible flexibility in a variety of situations.
