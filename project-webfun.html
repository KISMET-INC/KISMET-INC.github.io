Why didn't I use this portfolio instead?

    I went with a template so that I could get something nice to showcase my work up and running immediately not having to worry about designing and building out my own flashy sections and animations. I am still very pleased with the result and I may continue with it in the future but there was a reason I didn't jump on building this out fresh after graduation.

Technologies Used:

  • Used Javascript and JQuery to design and implement flashy animations and create an SPA which adds extremely smooth and visually pleasent transitions around the website.

  • Utilized the Bootstrap5 to implement information modals adding a classy modern feel ot the UX

  • CSS was used for styling, to add size , color and responsive design to the project making the UI visually appealing on a variety of device sizes
